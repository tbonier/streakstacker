# Streak Stacker

A lightweight, offline-first daily workout tracker that rewards consistency.

## Features
- One-tap "Mark Today Complete"
- Streaks & longest streak
- Points system with intensity/duration bonuses
- Weekly goal and progress bar
- 30-day calendar heat-like grid
- Badges & levels
- Notes + workout type & duration
- Export/Import JSON
- Optional daily reminder (browser notification)
- PWA: Install it like an app (mobile or desktop)
- 100% local â€” your data stays in your browser

## How to use
1. Open `index.html` in your browser.
2. Click **Mark Today Complete**, add minutes, intensity, type, and notes.
3. Set a **Weekly goal** (default 5 days).
4. (Optional) Set a **Daily reminder** time and allow notifications.
5. Export your data anytime from the header.

## Deploy
- Drop these files into any static host (GitHub Pages, Netlify, Vercel, S3/CloudFront).
